void HumDrum.Structures.Tree< T >.Coalesce ( Func< T, T, T, T >  coalescor,
Direction  firstArgument 
)
inline

Coalesces the tree from the bottom up. This will take a branch with 2 outer leaves, and perform an operation on them with the parent node's value, replacing it. This runs in place.

If a Node does not have 2 children, that part of the tree is ignored. This is only one iteration of Coalesce.

The value of the current node is considered in the third parameter of the coalescor.

Parameters
coalescorCoalescor - the function that takes 2 Nodes and returns a result
